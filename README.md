## 📝 About This Repository

This repository contains code examples developed for a university-level convergence informatics course using R, including:

- Improved R code files with **Doxygen-style headers** documenting purpose, usage, and structure
- Lecture demonstration codes covering unsupervised learning, dimension reduction, modeling, and supervised learning
- Assignment solutions for clustering, regression, classification, and DNA sequence analysis
- A **data/** directory containing sample datasets (TSV, CSV, XLSX) used in exercises
